# Proyecto 5: Homeland

## Edgar Cano by TripleTen.

## Descripción del proyecto

Este proyecto es una página web responsiva que muestra las ciudades natales de algunos empleados de TripleTen. El sitio fue desarrollado siguiendo un diseño proporcionado en Figma y se adapta correctamente a diferentes tamaños de pantalla: 1280px, 768px y 320px.

## Características del proyecto

- HTML5 semántico
- CSS3
- Flexbox
- Media Queries
- Metodología BEM (Flat)
- Normalización de estilos con normalize.css
- Fuentes locales
